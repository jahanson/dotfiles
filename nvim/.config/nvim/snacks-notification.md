# Snacks Picker Default Keybindings

## Navigation

| Key | Action |
|-----|--------|
| `j` / `<Down>` / `<C-n>` | Move down |
| `k` / `<Up>` / `<C-p>` | Move up |
| `gg` | Go to top |
| `G` | Go to bottom |
| `<C-d>` | Scroll list down |
| `<C-u>` | Scroll list up |
| `zz` / `zt` / `zb` | Scroll center/top/bottom |

## Selection & Actions

| Key | Action |
|-----|--------|
| `<CR>` / `<2-LeftMouse>` | Confirm/jump |
| `<Tab>` / `<S-Tab>` | Select item and move next/prev |
| `<C-a>` | Select all |
| `<C-q>` | Send to quickfix |

## Yank/Copy

| Key | Action |
|-----|--------|
| `y` | Yank item text to register (copies message) |

## Windows & Layout

| Key | Action |
|-----|--------|
| `/` | Toggle focus between input/list |
| `<Tab>` (in input) | Cycle windows |
| `<C-w>` | Pick window |
| `i` | Focus input |
| `<C-z>` | Toggle maximize |
| `<C-b>` | Toggle preview |
| `H` / `J` / `K` / `L` | Change layout (left/bottom/top/right) |

## Other

| Key | Action |
|-----|--------|
| `q` / `<Esc>` | Close/cancel |
| `?` | Toggle help |
| `<C-g>` | Inspect item |
| `<C-s>` | Open in split |
| `<C-v>` | Open in vsplit |
| `<C-t>` | Open in tab |
